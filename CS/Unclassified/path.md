

If you mean **“where am I in the filesystem?”**, use:

```bash
pwd
```

### 🔹 What it does

- Prints the **current working directory (absolute path)**
    

Example:

```bash
/Users/jeeva/projects/reputation-management
```

---

## 🔍 Related useful commands

### List files in current location

```bash
ls
```

### Show full path + contents

```bash
pwd && ls
```

### Move to another directory

```bash
cd <path>
```

---

## 🧠 If you meant something else

- **System location (hostname):**
    
    ```bash
    hostname
    ```
    
- **IP address:**
    
    ```bash
    ifconfig   # mac/linux
    ip addr    # linux
    ```
    
- **Geo location (approx via IP):**
    
    ```bash
    curl ipinfo.io
    ```
