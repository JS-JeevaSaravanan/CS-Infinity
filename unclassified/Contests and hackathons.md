# 🚀 Smart strategy to actually make money

### Phase 1 (0–2 months)

- Join **2–3 hackathons per month**
    
- Focus on:
    
    - simple working projects (not perfect)
        
    - team participation (very important)
        

### Phase 2 (2–6 months)

- Specialize (AI / Web / Data / Blockchain)
    
- Target **higher prize hackathons**
    
- Reuse ideas and improve them
    

### Phase 3 (6+ months)

- Turn hackathon projects →
    
    - startup ideas
        
    - freelance portfolio
        
    - SaaS products
        

---

# 🧠 Pro tips (this is what winners do)

- Don’t go solo → team up (design + dev + pitch)
    
- Pick **less crowded categories** (higher chance to win)
    
- Focus on **presentation + demo**, not just code
    
- Reuse past projects (huge advantage)
